# Hero Video Placeholder

Drop your cinematic background video here:

```
hero.mp4
```

## Requirements
- **Format:** MP4 (H.264 codec)
- **Resolution:** 1920×1080 or higher recommended
- **Duration:** Short loop (5–15 s works well)
- **Size:** Keep under ~15 MB for fast loading
- **Audio:** None required (video plays muted)

## Behaviour without the file
The landing page renders a premium CSS gradient fallback so the site
looks polished even before `hero.mp4` is added. Once the file is present,
`js/main.js` fades the video in over the gradient automatically.
